Interface IEventManager

Event manager for k8s.Models.Corev1Event objects. Contains various utility methods for emitting events on objects.

Create a IEventManager.AsyncStaticPublisher for a predefined event. The IEventManager.AsyncStaticPublisher is then called without any parameters. The predefined event is published with the initially given resource as the involved object.
